# TAKUMI CNC — ЗАДАНИЕ ДЛЯ ANTIGRAVITY

## БАЗА ПРОЕКТА
Внимательно изучи `GEMINI.md` — там описана вся эстетика, стиль (Apple + Япония + ЧПУ), шрифтовой контраст и технические решения.
Тексты, смыслы, программу и описание для кого этот курс — бери СТРОГО из приложенного файла `курс.txt`.

## ТВОЯ ЗАДАЧА
Создать современный, дорогой сайт для авторского курса.
Мы отказались от тяжелого 3D в пользу красивого видео, отказались от траурного ч/б в пользу гладкого темного/светлого фона с акцентами сакуры и инженерным "техно-фоном" (оси, формулы).

## КЛЮЧЕВЫЕ ТРЕБОВАНИЯ ДЛЯ РЕАЛИЗАЦИИ ПРЯМО СЕЙЧАС:
1. **Темы:** Реализовать переключатель Dark/Light mode.
2. **Типографика:** Настроить два шрифта (строгий основной + каллиграфический для цитат).
3. **Видео:** Сделать секцию для видео патрона. Настроить CSS-маску (растворение краев в фон) и привязать GSAP ScrollTrigger для зума при скролле.
4. **Supabase:** Подготовить форму входа/регистрации (Email) и скрипт блокировки доступа к папке `/lessons/` для неавторизованных пользователей.

Жду от тебя структуру проекта и первый вариант Главной страницы (`index.html` + `style.css` + `main.js`). Погнали!
