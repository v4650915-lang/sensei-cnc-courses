# 匠 SENSEI CNC

> *«Мастерство не сертифицируется. Оно передаётся.»*

Авторский онлайн-курс по программированию станков с ЧПУ на стойке **Fanuc**.  
Автор: **Владимир** — наладчик ЧПУ 5-го разряда, технолог-программист, 20+ лет в производстве.

---

## Стек технологий

| Инструмент | Назначение |
|---|---|
| Vanilla HTML / CSS / JS | Основа без фреймворков |
| GSAP 3.12+ | Все анимации |
| ScrollTrigger | Scroll-анимации |
| Lenis 1.1.9 | Плавный инерционный скролл |
| Supabase | Авторизация (Magic Link) + БД |

---

## Структура проекта

```
ПРОЕКТ КУРСЫ/
│
├── index.html          # Главная: Hero, видео патрона, о курсе, блок «Для кого»

│
├── style.css           # Единая таблица стилей (Dark / Light темы)
├── main.js             # Анимации GSAP, Lenis, переключатель тем, частицы сакуры
├── auth.js             # Supabase: Magic Link авторизация, роутинг, защита страниц
│
├── img/
│   ├── photo-author.jpg    # Фото Владимира у станка (страница «Об авторе»)

│
├── lessons/
│   ├── week1/index.html    # Материалы 1-й недели (закрыто — только после входа)
│   └── week2/index.html    # Материалы 2-й недели (закрыто — только после входа)

├── GEMINI.md           # Skill-файл для AI-ассистента (бренд, стек, дизайн)
```

---

## Публичная и закрытая зоны

**Открыто для всех:**  
`index.html`, `program.html`, `schedule.html`, `about.html`, `contacts.html`

**Только после авторизации:**  
`dashboard.html`, `lessons/week1/`, `lessons/week2/`  
Доступ проверяется через Supabase в `auth.js` — неавторизованные перенаправляются на главную.

---

## Запуск локально

```bash
npx http-server -p 8080
```
Открыть: `http://127.0.0.1:8080`

---

## Настройка Supabase

В файле `auth.js` заменить на реальные ключи проекта:

```javascript
const SUPABASE_URL = 'https://ВАШ_ПРОЕКТ.supabase.co';
const SUPABASE_ANON_KEY = 'ВАШ_АНОНИМНЫЙ_КЛЮЧ';
```

Без ключей авторизация и доступ к урокам работать не будут.
